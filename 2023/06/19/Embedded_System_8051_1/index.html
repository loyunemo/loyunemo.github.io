# 8051 指令系统和汇编程序设计

# 符号系统

符号对应内容备注
Rn8 组当前工作寄存器n=0-7，其中 0、1 可寄存器间接寻址
xxH内部 RAM 8 位地址00H-7FH, 特殊功能寄存器
#xxH8 位立即数指令中，data
#xxxxH16 位立即数指令中，data
xxxxHLCALL 和 LJMP 目的地址addr16，寻址空间位：64KB ROM
addr11ACALL 和 LJMP 目的地址addr, 目的地址放在与下条指令第一字节同一个 2KB 的 ROM 中
rel8 位带符号偏移量相对转移指令使用，包括条件转移和 SJMP
bit内部 RAM 和 SFR 位寻址空间的位地址
@前缀寄存器间接寻址或 (变址 + 基址) 寻址
/取反内部 RAM 和 SFR 位寻址空间的位地址前缀
(A)寄存器或者存储单元中内容*p
((A))内容作为地址再次指向内容**p
<-指令操作将右送到左寄存器或者 etc
^与|
/或^ 倒过来写
圈加异或

# 寻址指令长度
指令字节长度寻址空间
立即寻址双字节，三字节ROM
直接寻址双字节，三字节RAM 00H-7FH,SFR
寄存器寻址单字节Rn,A
寄存器间接寻址单字节指令RAM 00H-FFH,ROM 0000H-FFFFH
变址寻址单字节ROM 地址
相对寻址改 PC 指针实现程序转移
位寻址RAM 20H-2FH 128bit, 和 SFR 11 个可位寻址的 83bit
# 立即寻址


相当于 c 中的 (针对右边的操作数而言)

双字节或者三字节

# 直接寻址


相当于 c 中的取指 (针对右边的操作数而言)

# 寄存器寻址

指令高五位指令功能，低三位代表寄存器编号。

# 寄存器间接寻址

Ri,DPTR 可用于寄存器间接寻址，寄存器前面要加 @ 。

# 变址寻址

PC 或者 DPTR 作为基址寄存器，A 为变址寄存器，相加成为 16 位地址。

# 相对寻址

对 PC 转移偏移量 rel（-128-127）（补码表示，8 位带符号数）

# 位寻址

位地址表示方法：
1、直接使用位地址 D0H
2、单元地址加位 D0H.0
3、特殊功能寄存器 SFR 加位 PSW.0
4、位名称表示 P
