# 算符本征值

在量子力学中，乘上一个动量值可以看作对于一个函数进行了一次变换操作，类似的，量子力学的物理描述值被表示为算符。
可以记作：

p^=−iℏ∇\bold{\hat{p}}=-i\hbar\nablap^​=−iℏ∇
